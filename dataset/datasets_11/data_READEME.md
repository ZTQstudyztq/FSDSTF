## datasets_1.zip file description

This file is too large and has been split into multiple subvolumes with Bandizip. After downloading, follow these steps:

### Unzip method

1. Download ** all ** foldable files (' datasets_1.zip ', 'datasets_1.z01', 'datasets_1.z02'...)
2. Put all files in ** the same folder **
3. Right click on ** datasets_1.zip ** and select the unzip software (Bandizip is recommended)
4. Unzip it to get the full 'datasets_1.zip' file

### Considerations

- Missing any of the volumes will cause the decompression to fail
- Don't change the filename, or the decompressor won't recognize the order of the volumes
